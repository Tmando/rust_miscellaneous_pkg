Block-based version of the algorithm, which does not restrict pattern length.

This module implements the block-based version of the Myers pattern matching algorithm. It can be used for searching patterns of any length and obtaining semiglobal alignments of the hits. Apart from that, the Myers object in this module provides exactly the same API as the ‘simple’ version bio::pattern_matching::myers::Myers. For short patterns, the ‘simple’ version is still to be preferred, as the block-based algorithm is slower.
