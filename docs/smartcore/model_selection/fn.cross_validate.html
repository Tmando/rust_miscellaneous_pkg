Expand description

Evaluate an estimator by cross-validation using given metric.

  • fit_estimator - a fit function of an estimator
  • x - features, matrix of size NxM where N is number of samples and M is number of attributes.
  • y - target values, should be of size N
  • parameters - parameters of selected estimator. Use Default::default() for default parameters.
  • cv - the cross-validation splitting strategy, should be an instance of BaseKFold
  • score - a metric to use for evaluation, see metrics
